# Lunaa Paralanguage Sounds

Place your non-verbal audio files here. The system expects the following names:

- `sigh.mp3` - Triggered by `[tho-dai]`
- `chuckle.mp3` - Triggered by `[cuoi-khi]`
- `smile-chuckle.mp3` - Triggered by `[cuoi-nhe]`
- `tongue-click.mp3` - Triggered by `[tac-luoi]`
- `thinking.mp3` - Triggered by `[ngap-ngung]`

Format: MP3 or WAV recommended.
Volume: Keep them subtle and high quality for a therapeutic feel.
